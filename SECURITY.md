# Security Policy

Medomicile is a static GitHub Pages website. It does not collect passwords, payment data, medical records, or private patient files.

## Report a Security Issue

If you notice a security issue, do not publish sensitive details publicly. Contact the site owner directly through the phone or WhatsApp number displayed on the website.

## Scope

- Static pages: HTML, CSS, JavaScript
- Public assets and images
- GitHub Actions workflow for pharmacy duty data updates

## Notes

- The website uses a restrictive Content Security Policy.
- The pharmacy updater runs only on a schedule or manually from GitHub Actions.
- No secret keys or API tokens should be committed to this repository.
